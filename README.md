- **Universal Router** — Single entry point for swaps and liquidity
- **Real AMM Pools** — 0.3% fee, proper ERC20 transfers, slippage protection

- **Multi-Token Ready** — Deploy any ERC20 (MockERC20 included with 1M supply)
