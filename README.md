## Quick Start

` ` `bash
git clone https://github.com/GeraldNdlovu/tefa-dex.git
cd tefa-dex
npm install
cd frontend && npm install && cd ..
npm run dev
` ` `

## Commands

| Command | What it does |
|---------|--------------|
| `npm run dev` | Start the trading interface |
| `npx hardhat run scripts/add-liquidity.mjs --network sepolia` | Add liquidity |
| `npx hardhat run scripts/lp-breakdown.mjs --network sepolia` | Check your earnings |
| `npx hardhat run scripts/check-sepolia-balances.mjs --network sepolia` | Check balances |
